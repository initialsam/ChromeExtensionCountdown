# 倒數計時器國際化 (i18n) 支援

本擴充功能支援多國語言，會根據使用者瀏覽器的語言設定自動切換介面語言。

## 支援的語言

- 🇹🇼 繁體中文 (zh_TW) - 預設語言
- 🇺🇸 英文 (English) 
- 🇯🇵 日文 (日本語)
- 🇫🇷 法文 (Français)
- 🇪🇸 西班牙文 (Español)

## 語言自動偵測

擴充功能會自動偵測您的瀏覽器語言設定並顯示對應的介面語言：

- 如果瀏覽器語言設定為英文 → 顯示英文介面
- 如果瀏覽器語言設定為日文 → 顯示日文介面  
- 如果瀏覽器語言設定為法文 → 顯示法文介面
- 如果瀏覽器語言設定為西班牙文 → 顯示西班牙文介面
- 其他語言 → 顯示繁體中文介面（預設）

## 如何變更語言

### Chrome/Edge 瀏覽器

1. 開啟瀏覽器設定
2. 前往「語言」設定頁面
3. 調整語言的順序，將您想要的語言移到最上方
4. 重新載入擴充功能

### 詳細步驟（Chrome）

1. 點擊右上角的三個點 → 設定
2. 左側選單選擇「語言」
3. 點擊「語言」右側的「新增語言」來新增您想要的語言
4. 使用「⋮」按鈕將想要的語言移到清單最上方
5. 重新開啟擴充功能，介面語言即會改變

```
│   │   └── messages.json

│   ├── fr/             # 法文
│   │   └── messages.json
│   └── es/             # 西班牙文
│       └── messages.json
├── i18n.js             # i18n 輔助函式
├── manifest.json       # 使用 __MSG_*__ 宣告
└── ...
```

### 新增翻譯

如果您想要新增其他語言的翻譯：

1. 在 `_locales/` 目錄下建立新的語言資料夾（例如：`de` 代表德文）
2. 複製 `en/messages.json` 到新的資料夾
3. 翻譯 `messages.json` 中所有的 `message` 值
4. 擴充功能會自動支援新語言

### API 使用

擴充功能使用 Chrome Extension i18n API：

```javascript
// 取得翻譯文字
chrome.i18n.getMessage('keyName')

// 取得當前語言
chrome.i18n.getUILanguage()

// HTML 中使用
<span data-i18n="keyName">預設文字</span>
```

### i18n.js 輔助工具

提供便利的函式來簡化國際化：

- `I18N.getMessage(key)` - 取得翻譯文字
- `I18N.getLocale()` - 取得當前語言代碼
- `I18N.initPage()` - 自動初始化頁面所有 i18n 元素
- `I18N.getTimeText(seconds)` - 取得格式化的時間文字

## 翻譯品質

所有翻譯都經過仔細校對，確保：
- 文字準確且符合當地使用習慣
- 介面元素長度適當，不會造成排版問題
- 使用一致的術語和語氣

如果您發現任何翻譯問題，歡迎回報或提供改進建議！
