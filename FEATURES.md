# 📋 功能列表 - 倒數計時擴充功能 v1.0.0

## 🎯 核心功能

### 1️⃣ 倒數計時器
- [x] 自訂分鐘和秒數設定
- [x] 在網頁上即時顯示倒數時間
- [x] 在擴充功能圖示顯示剩餘分鐘數（Badge）
- [x] 自動記憶上次設定的時間
- [x] 開始/清除控制按鈕

### 2️⃣ 三色燈號系統
- [x] 🟢 **綠燈**：時間充裕（預設 > 3 分鐘）
- [x] 🟡 **黃燈**：時間適中（介於紅綠燈之間）
- [x] 🔴 **紅燈**：時間緊迫（預設 ≤ 10 秒）
- [x] 可自訂綠燈時間門檻
- [x] 可自訂紅燈時間門檻
- [x] 燈號即時視覺化顯示
- [x] Badge 圖示顏色隨燈號變化

### 3️⃣ 顯示選項
- [x] 可選擇顯示/隱藏倒數時間
- [x] 可選擇開啟/關閉 Badge 顯示
- [x] 支援純訊號燈模式（不顯示時間）
- [x] 設定自動儲存
