# 圖標說明

目前的圖標是簡單的佔位符。

## 使用沙漏 emoji ⌛ 作為圖標

我已經創建了 SVG 版本的圖標（icon16.svg, icon48.svg, icon128.svg），包含：
- 紫色漸變背景
- 白色沙漏 emoji ⌛

### 方法一：使用線上工具轉換
1. 訪問 https://svgtopng.com/ 或 https://cloudconvert.com/svg-to-png
2. 上傳 `icons/icon128.svg`、`icons/icon48.svg`、`icons/icon16.svg`
3. 轉換為對應尺寸的 PNG
4. 下載並替換現有的 PNG 文件

### 方法二：使用瀏覽器
1. 用 Chrome 打開任一 SVG 文件
2. 右鍵 > 另存圖片
3. 使用圖片編輯器調整尺寸
4. 保存為 PNG

### 方法三：使用設計軟體
使用 Figma、Photoshop、Illustrator 等：
1. 創建對應尺寸的畫布（16x16, 48x48, 128x128）
2. 添加圓形背景（漸變：#667eea 到 #764ba2）
3. 添加沙漏 emoji ⌛（白色）
4. 導出為 PNG

## 快速方法
目前 PNG 文件已經存在，可以直接使用。如果想要更好看的圖標：
- SVG 文件已生成在 icons 目錄
- Chrome Extension 也支援直接使用 SVG（Manifest V3）

## 測試
重新載入擴展後，工具列和擴展管理頁面會顯示新圖標。
