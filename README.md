# ⌛ 倒數計時 Chrome 擴充功能

## 版本資訊
**當前版本：1.0.0** (2025-12-26)

## 功能概述
一個功能強大且靈活的 Chrome 倒數計時器擴充功能，提供視覺化燈號提醒、網站自動計時規則，以及完整的設定管理功能。

---

## 📋 主要功能

### 1. 🕐 倒數計時器
- **自訂倒數時間**：支援分鐘和秒數設定
- **即時顯示**：在網頁上顯示倒數時間
- **Badge 顯示**：在擴充功能圖示顯示剩餘分鐘數
- **記憶上次設定**：自動記住最後使用的倒數時間

### 2. 🚦 三色燈號系統
- **綠燈**：時間充裕（可自訂門檻，預設 > 3 分鐘）
- **黃燈**：時間適中（介於紅燈與綠燈之間）
- **紅燈**：時間緊迫（可自訂門檻，預設 ≤ 10 秒）
- **自訂門檻**：可自由設定綠燈和紅燈的時間門檻
- **視覺化提醒**：在網頁上以明顯的燈號顯示時間狀態

### 3. 🌐 網站自動計時規則
- **規則管理介面**：專用的規則管理頁面
- **萬用字元支援**：使用 `*` 來匹配網址（例如：`*facebook.com*`）
- **自動觸發**：訪問指定網站時自動開始倒數
- **獨立設定**：每個網站可設定不同的倒數時間和燈號門檻
- **顯示選項**：可選擇顯示時間或僅顯示訊號燈
- **CRUD 功能**：
  - ➕ 新增規則
  - ✏️ 編輯規則
  - 🗑️ 刪除規則
  - 📋 查看所有規則

### 4. 🖱️ 右鍵選單快速新增
- **一鍵新增**：在任何網頁按右鍵選擇「新增此網站到倒數計時規則」
- **自動擷取**：自動識別當前網站的網址
- **智慧設定**：使用您上次的倒數時間作為預設值
- **即時通知**：新增成功後顯示系統通知

### 5. 💾 設定檔管理
- **匯出設定**：將所有設定匯出為 JSON 檔案
  - 倒數時間設定
  - 顯示選項
  - 燈號門檻
  - 所有網站規則
- **匯入設定**：從 JSON 檔案還原設定
- **備份與同步**：方便在不同裝置間同步設定
- **檔案命名**：自動以日期命名（例如：`countdown-settings-2025-12-26.json`）

### 6. 🎨 顯示選項
- **倒數時間顯示**：可選擇是否在網頁上顯示倒數時間
- **Badge 顯示**：可選擇是否在擴充功能圖示顯示剩餘分鐘
- **訊號燈模式**：可設定只顯示燈號，不顯示具體時間

### 7. 🎯 使用者體驗
- **漂亮的 UI**：漸層紫色背景，現代化設計
- **即時回饋**：操作後立即顯示狀態訊息
- **視覺動效**：按鈕 hover 效果、成功提示動畫
- **響應式設計**：自動調整介面大小
- **中文介面**：完整繁體中文支援

## 🚀 安裝方式

1. 下載或 clone 此專案
2. 開啟 Chrome 瀏覽器
3. 進入 `chrome://extensions/`
4. 開啟右上角的「開發人員模式」
5. 點擊「載入未封裝項目」
6. 選擇專案資料夾

---

## 📖 使用說明

### 基本使用
1. 點擊擴充功能圖示開啟 popup
2. 設定倒數時間（分鐘：秒數）
3. 調整顯示選項和燈號門檻（可選）
4. 點擊「開始」按鈕

### 設定網站規則
1. 點擊「管理網站規則」按鈕
2. 輸入網站網址（支援 `*` 萬用字元）
3. 設定倒數時間和燈號門檻
4. 點擊「新增規則」

### 快速新增規則（右鍵選單）
1. 在任何網頁按右鍵
2. 選擇「新增此網站到倒數計時規則」
3. 系統自動新增規則

### 編輯現有規則
1. 在規則列表中點擊「編輯」按鈕
2. 修改設定
3. 點擊「儲存修改」

### 匯出/匯入設定
1. 進入「管理網站規則」頁面
2. 捲動到「設定檔管理」區塊
3. 點擊「匯出設定」下載 JSON 檔案
4. 點擊「匯入設定」選擇檔案還原設定

## 🛠️ 技術規格

- **Manifest Version**: 3
- **權限**:
  - `storage`: 儲存設定和規則
  - `alarms`: 計時器功能
  - `contextMenus`: 右鍵選單
  - `activeTab`: 取得當前分頁資訊
  - `notifications`: 系統通知
  
- **檔案結構**:
  ```
  countdown/
  ├── manifest.json      # 擴充功能配置
  ├── background.js      # 背景服務（Badge 計時器、右鍵選單）
  ├── popup.html         # 主介面 HTML
  ├── popup.js           # 主介面邏輯
  ├── site-rules.html    # 規則管理頁面
  ├── site-rules.js      # 規則管理邏輯
  ├── content.js         # 內容腳本（網頁計時器）
  ├── content.css        # 內容腳本樣式
  └── icons/             # 圖示資料夾
  ```
