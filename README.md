# 倒數計時 Chrome Extension

一個功能完整的倒數計時器 Chrome 擴展程式。

## 功能特點

### 1. 靈活的時間設定
- **分鐘設定**：可選擇 +1、+5、+10、+30 分鐘
- **秒數設定**：可選擇 +1、+5、+10、+30 秒
- **累加功能**：所有時間按鈕都可累加，自由組合所需時間

### 2. 視覺化計時顯示
- **右下角訊號燈指示器**：
  - 🟢 綠燈：剩餘時間 > 3 分鐘
  - 🟡 黃燈：剩餘時間 ≤ 3 分鐘
  - 🔴 紅燈：時間到
- **即時倒數顯示**：以 MM:SS 格式顯示剩餘時間

### 3. 時間到提醒
- 時間到後等待 10 秒
- 10 秒後顯示**全螢幕提醒**
- 提供兩個選項：
  - ✅ **重新計時**：使用上次設定的時間重新開始
  - ❌ **關閉倒數計時**：完全停止計時器

### 4. 記憶功能
- 自動記錄最後一次設定的時間
- 下次開啟時自動載入上次的設定

## 安裝方式

1. 開啟 Chrome 瀏覽器
2. 進入 `chrome://extensions/`
3. 開啟右上角的「開發人員模式」
4. 點擊「載入未封裝項目」
5. 選擇此專案的資料夾

## 使用方法

1. 點擊瀏覽器工具列上的擴展圖示
2. 使用「+分鐘」和「+秒數」按鈕設定時間
3. 點擊「開始」按鈕啟動計時
4. 計時器會顯示在網頁右下角
5. 時間到後會有視覺和全螢幕提醒

## 文件結構

```
countdown/
├── manifest.json         # 擴展配置文件
├── popup.html           # 彈出視窗 HTML
├── popup.js             # 彈出視窗邏輯
├── content.js           # 內容腳本（計時器邏輯）
├── content.css          # 內容腳本樣式
├── icons/               # 圖標資料夾
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── README.md            # 說明文件
```

## 技術細節

- **Manifest V3**：使用最新的 Chrome Extension API
- **Chrome Storage API**：用於儲存用戶設定
- **Content Scripts**：在網頁中注入計時器 UI
- **Message Passing**：popup 與 content script 之間的通訊

## 注意事項

- 計時器會在當前活躍的分頁中顯示
- 切換分頁不會影響計時
- 關閉分頁會停止該分頁的計時器
- 圖標為簡單的佔位符，建議替換為自訂設計
