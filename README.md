## 💭 作者的話

你是否有過這樣的經驗？

*「只是想休息一下，打開 Facebook、Instagram 看看朋友動態......」*  
*「怎麼不知不覺就過了 30 分鐘？」*  
*「明明只想看一下，結果停不下來了！」*

我也是。因此我開發了這個倒數計時器。

這不是一個「封鎖網站」的工具，而是一個**提醒工具**。它會在你設定的時間內，用視覺化的燈號提醒你：「嘿，你已經在這個網站待多久了？」

- 🟢 綠燈：時間還很充裕，放鬆享受吧
- 🟡 黃燈：注意一下時間囉
- 🔴 紅燈：該停下來了

透過這樣的方式，讓我們對自己的時間使用更有**意識**和**控制力**。不是強迫你離開，而是溫柔地提醒你：「該做正事了」。

希望這個工具也能幫助你，找回對時間的掌控。

## 📋 主要功能

### 🕐 倒數計時器
- 自訂倒數時間（分鐘 + 秒數）
- 網頁上即時顯示剩餘時間
- 擴充功能圖示顯示剩餘分鐘數
- 自動記憶上次設定

### 🚦 三色燈號系統
- **🟢 綠燈**：時間充裕（可自訂，預設 > 3 分鐘）
- **🟡 黃燈**：時間適中（介於紅燈與綠燈之間）
- **🔴 紅燈**：時間緊迫（可自訂，預設 ≤ 10 秒）

### 🌐 網站自動計時規則
- 為特定網站設定自動倒數計時
- 支援萬用字元（如 `*facebook.com*`、`*instagram.com*`）
- 每個網站獨立設定時間和燈號門檻
- 完整的規則管理（新增、編輯、刪除）

### 🖱️ 右鍵快速新增
- 在任何網頁右鍵選擇「新增此網站到倒數計時規則」
- 自動擷取當前網站網址
- 智慧防止重複規則

### 💾 設定管理
- 匯出所有設定為 JSON 檔案
- 匯入設定快速還原
- 方便備份和跨裝置同步

## 🚀 安裝方式

### 開發者模式安裝
1. 下載或 clone 此專案
2. 開啟 Chrome，進入 `chrome://extensions/`
3. 開啟「開發人員模式」
4. 點擊「載入未封裝項目」
5. 選擇專案資料夾

---

## 📖 快速開始

### 1️⃣ 基本使用
```
1. 點擊擴充功能圖示
2. 設定倒數時間（例如：10 分鐘）
3. 點擊「開始」
4. 在網頁上看到倒數計時和燈號
```

### 2️⃣ 為 Facebook 設定自動計時
```
1. 點擊「管理網站規則」
2. 輸入網址：*facebook.com*
3. 設定倒數時間：5 分鐘
4. 點擊「新增規則」
5. 下次訪問 Facebook 會自動開始倒數
```

### 3️⃣ 切換語言
```
1. 開啟規則管理頁面
2. 捲動到底部「🌐 語言設定」
3. 選擇你想要的語言
4. 立即生效！
```

## 🛠️ 技術資訊

### 技術棧
- Chrome Extension Manifest V3
- Vanilla JavaScript (ES6+)
- Chrome APIs: Storage, Alarms, Context Menus, Notifications
- Dynamic i18n System with Fetch API

### 專案結構
```
countdown/
├── manifest.json           # 擴充功能配置
├── background.js           # 背景服務
├── popup.html/js           # 主介面
├── site-rules.html/js      # 規則管理
├── content.js/css          # 網頁注入腳本
├── i18n.js                 # 動態 i18n 系統
├── _locales/               # 多語言翻譯檔案
│   ├── en/
│   ├── zh_TW/
│   ├── ja/
│   ├── fr/
│   └── es/
└── icons/                  # 圖示
```
